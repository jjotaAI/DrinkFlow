# DrinkFlow
*Aplicação de gestão de vendas de uma rede de bebidas alcoólicas*
> Copie o arquivo .env.example para .env e preencha com suas credenciais antes de rodar a aplicação.
> Atenção! Apenas o DB_USER e DB_PASSWORD devem ser alterados no momento da criação do arquivo .env e o arquivo .env deve ser criado na raiz do projeto (IMPORTANTE), no mesmo escopo em que está criado o .env.example!
